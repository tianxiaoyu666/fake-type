# Fake Type - 演示神器

> 职业院校技能大赛专用神器 —— 毕竟现在比的不是技能，是表演。

## 这是什么？

懂的都懂。

职业院校技能大赛，名义上是"技能"大赛，实际上呢？提前拿到题目、反复排练、背诵代码、表演敲键盘... 说白了就是 **PPT 表演大赛**。

既然大家都在演，那就演得专业一点。

本插件让你：
- 乱敲键盘也能输出完美代码
- 不用背代码，专心表演就行
- 终端输出也能提前准备好
- 评委看了直呼内行

**你只需要负责表演，代码的事交给我。**

## 为什么会有这个项目？

因为现实就是这么魔幻：

1. **提前知道题目** - 比赛前一个月就开始"针对性训练"了，懂？
2. **反复排练** - 不是在学技术，是在背代码、练手速
3. **评委看的是什么？** - 你敲得快不快、流不流畅、有没有卡壳
4. **真正的技术能力？** - 不好意思，不在评分标准里

所以与其花时间背代码，不如用这个插件，把精力放在"表演"上。

**毕竟，卷不过就加入嘛。**

别跟我提创新大赛，更魔幻。

表面上喊着"创新创业"，实际上呢？

## 功能特性

### 假装打代码
- 预先准备好代码内容
- 在目标文件中随便敲键盘，每按一个键显示一个预设字符
- 支持多文件独立映射
- 回车、Tab 等特殊键正常工作
- 退格键同步回退（万一演砸了还能补救）

### 假终端输出
- 自定义终端前缀（完美模拟 PowerShell/Bash）
- 预设终端命令和输出结果
- 乱敲键盘逐字符显示预设内容
- 评委根本看不出来

### 数据持久化
- 映射关系自动保存
- 重启编辑器后自动恢复
- 进度实时保存（中途关机也不怕）

## 安装方式

### 方式一：从 VSIX 安装（推荐）
1. 下载 `.vsix` 文件
2. 在 VS Code/Cursor 中按 `Ctrl+Shift+X` 打开扩展面板
3. 点击右上角 `...` → "从 VSIX 安装..."
4. 选择下载的文件
5. 开始你的表演

### 方式二：从源码构建（适合想装逼的）

```bash
git clone https://github.com/tianxiaoyu666/fake-type.git
cd fake-type
npm install
npm run compile
vsce package --allow-missing-repository
```

## 使用方法

### 代码映射

1. 点击左侧活动栏的 **Fake Type** 图标
2. 在"代码映射"区域：
   - 点击"选择"按钮选择目标文件
   - 粘贴你要"表演"的代码
   - 点击"保存"
3. 打开目标文件，随便敲键盘
4. 看着完美的代码一个字一个字出现
5. 评委：这小伙子代码写得真熟练啊

### 终端映射

1. 在"终端命令"区域：
   - 设置终端前缀（如 `PS C:\Users\xxx> `）
   - 粘贴完整的终端内容（包括命令和输出）
   - 点击"添加"
2. 点击"打开演示终端"
3. 随便敲键盘，看着命令和输出完美呈现
4. 评委：这小伙子对命令行很熟练啊

## 快捷命令

| 命令 | 说明 |
|------|------|
| `Fake Type: 切换启用/禁用` | 暂停/启用功能（万一评委要凑近看） |
| `Fake Type: 从剪贴板加载` | 快速为当前文件加载内容 |
| `Fake Type: 重置到开头` | 重置当前文件进度（再演一遍） |
| `Fake Type: 打开演示终端` | 打开演示用的伪终端 |
